3RESULTS & PRIORITY RANKING

SCORING METHODOLOGY

Each facility is scored across 4 dimensions, each contributing 25 points (max total: 100):

1. Reporting Rate (25 points)

Facility Score = (Months Reported / Expected Months) x 25

Expected Months = 12 x Number of years in data.

2. Coherency (25 points)

Score = (Checks Passed / Total Checks) x 25

Four rules: Suspected <= OPD, Tested <= Suspected, Confirmed <= Tested, Confirmed <= Treated

3. Indicator Quality (25 points)

Score = ((Total Records - Issue Records) / Total Records) x 25

4. Outlier Score (25 points)

Score = ((Total Values - Outlier Values) / Total Values) x 25

Priority Classification

  • HIGH PRIORITY Total Score < 75 (requires immediate attention)
  • MEDIUM PRIORITY Total Score 75-89 (needs improvement)
  • LOW PRIORITY Total Score >= 90 (acceptable data quality)
