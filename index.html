3RESULTS & PRIORITY RANKING

SCORING METHODOLOGY

Each facility is scored across 4 dimensions, each contributing 25 points (max total: 100):

1. Reporting Rate (25 points)

Facility Score = (Months Reported / Expected Months) × 25

Expected Months = 12 × Number of years in data. A month is "reported" if at least 1 key variable has a value.

Example: Data spans 3 years → Expected = 36 months. Facility reported 30 months → Score = (30/36) × 25 = 20.8 points

2. Coherency (25 points)

Score = (Checks Passed / Total Checks) × 25

Four rules: Suspected ≤ OPD, Tested ≤ Suspected, Confirmed ≤ Tested, Confirmed ≤ Treated

Example: 100 records, 90 pass all checks → Score = (90/100) × 25 = 22.5 points

3. Indicator Quality (25 points)

Score = ((Total Records - Issue Records) / Total Records) × 25

Inappropriate Reporting: Facilities that should NOT report certain variables but have values.

Missing Required Reporting: Facilities that SHOULD report certain variables but have missing values. If ANY selected variable is missing, it's an issue.

Incomplete Key Indicators: Must have ALL 5 key indicators (OPD, Suspected, Tested, Confirmed, Treated). 3/5 or 4/5 = Issue. Only 5/5 = OK.

4. Outlier Score (25 points)

Score = ((Total Values - Outlier Values) / Total Values) × 25

IQR Method: Q1 (25th %), Q3 (75th %), IQR = Q3-Q1. Bounds: Q1-1.5×IQR to Q3+1.5×IQR. Values outside = outliers.

Example: Q1=10, Q3=50, IQR=40. Bounds: [-50, 110]. Value 150 = outlier.

Summary Metrics

Average Reporting Rate = Sum of all facility reporting rates / Number of facilities
Average DQA Score = Sum of all facility total scores / Number of facilities

Priority Classification

  • HIGH PRIORITY Total Score < 75 (requires immediate attention)
  • MEDIUM PRIORITY Total Score 75-89 (needs improvement)
  • LOW PRIORITY Total Score ≥ 90 (acceptable data quality)
